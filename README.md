# medical-ECG-learning-app-HUKM (Internship Project)
A diagnostic and educational tool to help doctors and medical students to analyze ECG graphs and receive disease predictions. It also serves as an ECG learning platform for medical students and medical staffs.

## Overview
This project was developed during my diploma internship in IT.
It is a medical learning application designed for HUKM students and medical staff to enhance learning through interactive modules, multimedia resources, and practical examples.
The application combines structured lessons, quizzes, scenario-based learning slides, video content, and ECG samples to provide a comprehensive medical learning experience.

Due to confidentiality and institutional policy, the source code is not publicly shared.

## Features
- ECG learning exercises 
- Quizzes and assessments
- Interactive learning slides
- Video resources

## Technology Stack
- Frontend: Flutter (Dart)
- Backend & Database: None (all data and content stored locally within the app)

## Testing & Validation
- Application was tested by medical doctor and medical students
- Feedback was collected and improvements were made
